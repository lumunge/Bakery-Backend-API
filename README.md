### BAKERY SHOP BBAKEND API

### ABOUT

An api for a bakery frontend webapp that serves data and handles CRUD operations
and user authentication

### TECHNOLOGIES:

    * cors
    * express
    * bcrypt
    * json web tokens
    * mongoose and mongodb
    * Google-Maps-React
    * short id

### FEATURES

    * Create New Products
    * Get Products
    * User Authentication
    * User authentication
